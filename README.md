# QuizBuddy
Live project
https://quizzbuddyy.herokuapp.com/

## For local setup run:-
* pip install -r requirements.txt
* python manage.py migrate
* python manage.py runserver

## Run tests:-
* python manage.py  test

## Environment variables:-
* Database
  * DATABASE_URL
* Google authentication
  * GOOGLE_CLIENT_ID
  * GOOGLE_CLIENT_SECRET
* Caching
  * REDIS_URL
